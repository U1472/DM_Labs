# Лабораторная работа №4 «Поиск ассоциативных правил в транзакционных данных»
## Цель работы:
Освоить методы поиска ассоциативных правил, реализовать ключевые алгоритмы (Apriori, FP‑Growth), применить их к реальным и синтетическим наборам данных, провести сравнительный анализ производительности и интерпретировать полученные правила.
## Задания:
Необходимо реализовать оба алгоритма (Apriori и FP‑Growth) «с нуля» без использования готовых библиотек типа mlxtend, pyfpgrowth или efficient_apriori.
Допускается использовать pandas, numpy, itertools, collections, а также стандартные структуры данных.
## Технические требования
- Язык реализации: Python 3.8+.
- Библиотеки: разрешены pandas, numpy, matplotlib, seaborn, networkx, scipy, time,
itertools, collections.
- Запрещены готовые реализации Apriori/FP-Growth
(например, mlxtend.frequent_patterns, pyfpgrowth, efficient_apriori).
- Код должен быть задокументирован (docstrings, комментарии к ключевым
моментам).
- Результаты экспериментов должны быть воспроизводимы (фиксированный
random_seed для синтетики).
