# Главная ветвь в DM_Labs
Данный репозиторий сделан для работ по дисциплине *"Интеллектуальный анализ данных"*
Каждая лабораторная работа (за исключением 1.1) будет находиться на *своей ветке проекта* (работа 1.1 находиться в ветке работы 1)

## Данные в датасете
- date: Дата записи
- totalChallan: Общая сумма штрафов, выписанных за день.
- disposedChallan: Штрафы, оплаченные/урегулированные гражданами.
- pendingChallan: Неоплаченные/непогашенные штрафы.
- pendingAmount: Сумма непогашенного штрафа (в индийских рупиях).
- disposedAmount: Сумма собранного штрафа (в индийских рупиях).
- totalAmount: Общая сумма начисленных штрафов (в индийских рупиях).
- pendingCourt: Дела, находящиеся на рассмотрении в суде.
- disposedCourt: Дела, разрешенные в суде.
- totalCourt: Всего дел, переданных в суд.
